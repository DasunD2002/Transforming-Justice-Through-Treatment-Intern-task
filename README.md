This is a static website for Baltimore Therapeutic Treatment Center (BTCC). It shows the mission, latest events, news, and contact information.
The website is built with HTML, CSS, and JavaScript, and it is fully responsive for desktops, tablets, and mobile devices.

Usage Instructions

Navigation: Use the top menu to go to sections like Home, About, Opportunity, and Contact.
Sign Up Buttons: Click the "Sign up" buttons to simulate user sign-up (alerts appear).
Events Section: Hover over event cards to see a highlight effect.
Newsletter: Enter an email in the footer subscription form to test subscription alert.

Responsive Design

The website adapts to different screen sizes:
Desktop: Large images, multi-column layouts.
Tablet: Adjusted grids and smaller images.
Mobile: Single-column layouts, resized images, and simplified navigation.


Dependencies
Boxicons for icons.
Modern browser (Chrome, Firefox, Edge, Safari).
